# Forma Studio — Digital Marketing Agency Website

A single-page marketing agency website with an AI-powered content generator.

## Deploying to Cloudflare Pages

1. Push this repository to GitHub
2. Go to [Cloudflare Pages](https://pages.cloudflare.com)
3. Click **Create a project** → **Connect to Git**
4. Select this repository
5. Build settings:
   - **Framework preset:** None
   - **Build command:** (leave empty)
   - **Build output directory:** `/` (just a slash)
6. Click **Save and Deploy**

That's it. Your site will be live at `your-project.pages.dev`.

## Files

- `index.html` — the full website + AI content generator
- `_redirects` — Cloudflare routing rule
- `README.md` — this file
